"Data never comes clean ( if you get it neat, you are lucky )."
Through the process of data cleaning we will be able to detect and correct the corrupt
or inaccurate records and tables, find the missing fields and replace them followed by
sorting the required files and filtering the unwanted to prepare a complete usable
dataset for analysis.
Stage 1
Clean
Through the process of analysing the data, we will be able to define the problem
statements and the intention behind creating a particular visualization.
Many possible tangents around the dataset could also be discovered which
can make the narrative strong and make the insights meaningful.
Stage 2
Analyze
From the finalized narrative start building a static visualization concept initally.
working on interaction without finalizing the flow with effect the process in the
later. After building a visualization with the proper design language interaction
can then be added if necessary.
Stage 4
Visualize
From the Ideas and Insights comes concepts and narratives. In most of the cases
the narrative comes only after we analyze data. Verification of the data with
the external sources ( news articles, encyclopedia) will not only provide a strong
assurity to the narrative but also build confidence within to move forward.
